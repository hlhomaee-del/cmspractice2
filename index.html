The Four Code Sets

ICD-10-CM

Diagnosis codes. Tells the payer WHY the service was performed. Supports medical necessity.

Example: J06.9 (Acute upper respiratory infection)

CPT

Procedure/service codes. Tells the payer WHAT was done. Professional services.

Example: 99213 (Office visit, established patient)

HCPCS Level II

Supplies, equipment, drugs, transportation, and services not covered by CPT.

Example: A4253 (Blood glucose test strip)

ICD-10-PCS

Hospital inpatient procedures only. NOT used in physician/outpatient billing.

Example: 0DBN0ZZ (Excision of sigmoid colon)

SOAP Note Structure

S – Subjective
What the patient reports: symptoms, history, complaints
O – Objective
Provider findings: exam results, vitals, test results
A – Assessment
The provider's diagnosis or clinical impression
P – Plan
Treatment plan: medications, follow-up, referrals
Key Coding Rule
The Assessment section contains the diagnosis. Coders look here first for the confirmed diagnosis to code. The Plan supports what services/procedures were ordered.

The Key Distinction

Confirmed Diagnosis

Provider states a definitive condition in the Assessment. Code this.

Example: "Type 2 diabetes mellitus"

Symptom

A complaint or finding without a confirmed cause. Code ONLY if no confirmed diagnosis explains it.

Example: "Headache" (when no underlying cause found)

Service / Procedure

What was done for the patient. Coded with CPT or HCPCS. Goes in Item 24D.

Example: 99213 (Office visit)

How They Connect on a Claim
ICD-10-CM (Item 21) explains WHY → CPT/HCPCS (Item 24D) shows WHAT → Item 24E links them with a diagnosis pointer.

Key Distinction: Rendering vs Billing Provider

Rendering Provider (24J): The individual clinician who directly provided the service (Dr. Maria Chen).

Billing Provider (Item 33): The entity submitting the claim — typically the practice or facility (Springfield Family Medicine).

Outpatient Rule

Do NOT code "rule out" or "suspected" as confirmed in outpatient

Code to Specificity

Always choose the most specific code supported by documentation

🎓 "If You See This, Think This" Quick Guide

Sore throat + positive strep test → Code streptococcal pharyngitis (J02.0), not symptom

"Rule out MI" → Outpatient: code the symptom (chest pain), NOT MI

Assessment says "Type 2 DM" → E11.9 (without complications if none documented)

Item 24E shows "A" → This service line is linked to the FIRST diagnosis in Item 21

Two NPIs on the form → Rendering (24J) = did the work · Billing (33a) = submits the claim
